핵심 요구사항 분석

  • Cinematic & Non-Verbal UI — 텍스트 의존도 10% 미만, 시각적 애니메이션·게이지·햅틱 피드백 중심의 몰입형 UX
  • 3D Galaxy 뷰 — 아티스트별 행성이 3D 궤도를 도는 메인 화면, 실시간 FPI 지수에 따라 크기·텍스처 가변
  • POP 토큰 이코노미 — 걷기(Health API) + YouTube 스밍으로 POP 획득, Zap 부스터로 행성 성장, 이클립스 방어전
  • 크라우드 펀딩 (Archive) — 팬 주도 서포트 캠페인 개설, 목표 미달 시 100% 자동 환불, 결과 보고 공유
  • 현상금 게시판 (K-Inside) — POP 기반 신뢰 소통, 채택 시 POP 즉시 전송
  • 진화형 디지털 신분증 — 기여도에 따라 유리→황금→심연으로 실시간 진화하는 카드 UI
  • 수익화 — No-Force 광고 정책(자발적 리워드 광고), 프리미엄 구독, 글로벌 부스트 앰플
  • 어뷰징 방지 — YouTube 90% 이상 시청 검증, 포그라운드 체크, 일일 시청 횟수 제한

제안 포인트

1. 센서 기반 게이미피케이션 직접 구축 경험

자체 프로덕트 DayStarter에서 ML Kit 얼굴 인식·조도 센서·배터리 센서 기반의 수면 인증 시스템과 듀얼 토큰 이코노미, 7단계 루트박스, Spine 2D 캐릭터 시스템을 직접 설계·구현한 경험이 있습니다. 본 프로젝트의 POP 이코노미, Health API 연동, 행동 기반 리워드 시스템과 높은 유사도를 가집니다.

2. Firebase 풀스택 + 광고 수익화 경험

Fortune App에서 Firebase(Auth, Analytics, Crashlytics, Remote Config) 기반 풀스택 앱을 3개월 내 iOS/Android 동시 출시하였으며, 자체 광고 SDK를 개발하여 앱 업데이트 없이 광고 전략을 변경할 수 있는 시스템을 구축한 경험이 있습니다.

3. React Native 크로스플랫폼 실전 경험

Pilates App에서 React Native 기반 iOS/Android/Web 3개 플랫폼을 4개월 만에 동시 출시한 경험이 있으며, 예약·결제·CRM·커뮤니티·채팅 등 8개 기능 모듈을 올인원으로 구현한 실적이 있습니다.

4. 2단계 납품 전략으로 빠른 시장 검증

Phase 1(90일)에서 Galaxy 뷰, Home(퀘스트/만보기/YouTube), My Page 등 핵심 3탭 MVP를 먼저 출시하여 시장 반응을 검증하고, Phase 2(30일)에서 Archive, K-Inside, 수익화 모듈을 확장하는 Lean 전략을 제안합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

단순 노동형 덕질을 넘어, 팬덤의 연대와 위기 극복을 통한 '성취' 중심의 글로벌 생태계를 구현하는 차세대 시네마틱 글로벌 팬덤 플랫폼의 MVP를 개발합니다. 클라이언트가 제공하는 상세 PRD(10개 이상 섹션)와 React/Tailwind 디자인 코드, 3D 개발 소스를 기반으로 기술 구현에 집중하여 빠른 시장 검증(Lean)을 목표로 합니다.

핵심 목표: 전 세계 유저가 직관적으로 소통하고 몰입할 수 있도록 시각적 애니메이션, 게이지, 햅틱 피드백을 활용한 Cinematic & Non-Verbal UI를 구현합니다.

2. 사용자 정의

서포트 주최자팬덤 활동을 조직하는 리더Archive 캠페인 개설·운영, 결과 보고 공유

3. 기능 요구사항

3.1 사용자 인증 (Phase 1)

  • [필수] Firebase Auth 기반 이메일/비밀번호 회원가입 및 로그인
  • [필수] 소셜 로그인 (Google, Apple)
  • [필수] 온보딩 플로우 (아티스트 선택, 프로필 설정)

3.2 Galaxy & Social Proof — 메인 화면 (Phase 1)

  • [필수] 3D 은하계 뷰: 아티스트별 행성이 궤도를 도는 렌더링 (클라이언트 3D 소스 활용)
  • [필수] 실시간 FPI 지수에 따라 행성 크기·표면 텍스처(금속, 보석 등) 가변
  • [필수] 이클립스 이벤트: 붉은 경고광 + 행성 축소 위기 자동 발생
  • [필수] 라이브 소셜 프루프: 슈퍼 부스트 시 섬광 효과 + 공지 배너 실시간 노출

3.3 Home — 활동 센터 (Phase 1)

  • [필수] Today's Quest: 만보기 미션, 출석 체크, YouTube 음원 스밍 미션
  • [필수] Health API 연동: OS 수준 걸음 수 백그라운드 트래킹
  • [필수] YouTube IFrame API: 시청 시간 트래킹, 90% 이상 시청 시 리워드, 포그라운드 체크
  • [Phase 2] AI Daily Briefing: 글로벌 팬덤 뉴스 카드 요약
  • [Phase 2] 긴급 돌발 퀘스트: 타임어택 미션

3.4 Archive — 팬덤 펀딩 (Phase 2)

  • [필수] 팬 주도 서포트 캠페인 개설 (전광판, 커피차 등)
  • [필수] D-Day까지 목표 미달성 시 100% 자동 환불
  • [필수] 성공 시 결과 사진·소감 등록 및 전체 팬덤 공유

3.5 K-Inside — 지식인 게시판 (Phase 2)

  • [필수] POP 현상금 질문 등록
  • [필수] 답변 채택 시 POP 즉시 전송

3.6 My Page — 진화형 디지털 신분증 (Phase 1)

  • [필수] Evolutionary ID Card: 유리(Glass) → 황금(Gold) → 심연(Obsidian) 실시간 진화
  • [필수] 통계 시각화: 누적 걸음 수, 참여 캠페인 수, 총 기여 POP

3.7 수익화 (Phase 2)

  • [필수] AdMob 리워드 광고: 퀘스트 완료 시 자발적 시청 → 3배 보상, 일일 횟수 제한
  • [필수] 프리미엄 구독: 광고 제거 월간 정기 구독
  • [필수] 글로벌 부스트 앰플: 전체 팬덤 POP 획득량 1.5배 증가 (1시간)

3.8 마케팅 웹 (Phase 2)

  • [필수] 읽기 전용 모바일 웹: 앱 소개, Galaxy 뷰 미리보기, 앱 다운로드 유도
